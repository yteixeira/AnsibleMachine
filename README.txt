Below are the steps describing how to get the Ansible Multivagrant App up and running on your machine.

1. Download all files from this repository into a new folder.
2. If you are using Windows, go into that folder via Gitbash (making sure you are running Gitbash as Administrator).
3. Type "vagrant up" into the terminal and hit enter, this shall start the application.
4. Once the script has finished running, and the [App Running] task is shown with a flashing cursor below, you can access the app in your browser.
